## Мова програмування Удав

### Інсталяція

Щоб запустити інтерпретатор Удава, вам необхідно буде виконати один з наступних варіантів:

___1. Скомпілювати інтерпретатор___ (це дасть вам більше можливостей у майбутньому)

___2. Запустити інтерпретатор без компіляції___ (це також підійде, якщо ви просто хочете спробувати мову Удав і не хочете встановлювати або компілювати файли)

***1. Компіляція інтерпретатора***

Компілювання інтерпретатора необхідно для отримання виконуваного файлу udav. Для цього виконайте наступні кроки:

1. Клонуйте репозиторій з GitHub:

``` bash
git clone https://github.com/NikitaBerezhnyj/Udav.git
```

2. Перейдіть в теку з проектом:

``` bash
cd Udav
```

3. Встановіть пакет pyinstaller:

``` bash
pip install pyinstaller
```

4. Запустіть компіляцію:

``` bash
pyinstaller -F -n udav shell.py
```

Після завершення компіляції в теці з проектом буде створений виконуваний файл udav. Після ініціалізації цей файл можна буде використовувати для запуску програм мовою Удав з будь-якої теки.

***2. Запустити інтерпретатор без компіляції***

Щоб запустити інтерпретатор без компіляції, виконайте наступну команду в теці з проектом:

1. Клонуйте репозиторій з GitHub:

``` bash
git clone https://github.com/NikitaBerezhnyj/Udav.git
```

2. Перейдіть в теку з проектом:

``` bash
cd Udav
```

3. Запуск інтерпретатора

``` bash
python shell.py
```

Ця команда запустить інтерпретатор Удав в теці з проектом. Ви зможете написати і запустити програму, не компілюючи інтерпретатор.

#### Ініціалізація

Ініціалізація інтерпретатора Удав необхідна для того, щоб можна було використовувати його з будь-якої теки на вашому комп'ютері. Для цього виконайте наступні дії:
1. Перейдіть у теку із скомпільованим інтерпретатором

``` bash
cd dist
```

2. Запустіть файл інтерпретатора

``` bash
./udav
```

3. Використайте наступну команду:
   
``` python
ініціалізація
```

4. Завершіть роботу інтерпретатора:

``` python
вихід
```

Після ініціалізації інтерпретатора файл udav буде скопійовано в теку з іншими утилітами, що дасть можливість використовувати його для запуску програм мовою Удав з будь-якої теки.

#### Деніціалізація

Деініціалізація інтерпретатора Удав необхідна, якщо ви більше не плануєте використовувати його. Для цього виконайте наступні кроки:

1. Відкрийте інтерпретатор

``` bash
udav
```

2. Використайте наступну команду:
   
``` python
деініціалізація
```

3. Завершіть роботу інтерпретатора:

``` python
вихід
```

Це видалить Удав з теки з утилітами, але залишить його у теці з проєктом, тому ви зможете використовувати його і потім, якщо забажаєте

***Умови if (якщо), else if (інакше_якщо) та else (інакше)***

``` python

# Умова якщо-інакше повний запис
якщо а < 5 тоді
    друк("<5") 
інакше друк(">8")

# Умова якщо-інакше_якщо-інакше запис в один рядок
якщо а < 5 тоді; друк("<5") інакше_якщо а > 5 і а < 8 тоді; друк("<5 and >8") інакше друк(">8")

# Умова якщо-інакше_якщо-інакше повний запис
якщо а < 5 тоді 
    друк("<5") 
інакше_якщо а > 5 і а < 8 тоді 
    друк("<5 and >8") 
інакше друк(">8")
```

### Команди інтерпретатора

- ***допомога*** - показує всі доступні команди для терміналу інтерпретатора та базовий синтаксис мови

- ***запуск*** - запускає файл .udav (приклад використання: ***запуск("приклад.udav")***)

- ***цт*** - ця тека. Показує шлях до поточної теки

- ***лф*** - лист файлів. Показує вміст поточної теки

- ***зт*** - змінити теку. Змінює поточну теку

- ***сф*** - створити файл. Створює файл у поточній теці (приклад використання: ***сф приклад.txt***, якщо розширення не вказане, то воно буде за замовчуванням встановлене, як .udav)

- ***ст*** - створити теку. Створює теку у поточній (приклад використання: ***ст тестова_тека***)

- ***очистити*** - очищує термінал інтерпретатора

- ***вихід*** - закриває інтерпретатор

- ***версія*** - показує поточну версію Удав, що встановлена на вашому ПК

- ***УДАВ*** - виводить основну інформацію про стан Удава
